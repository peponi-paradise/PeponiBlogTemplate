## 1. Introduction

- 글에 헤더를 사용하는 경우, 헤더에 원하는 ID를 부여하는 기능
- `## 헤더 이름 {#ID}{:md}`로 부여한다.
- 마크다운에서는 대체로 ID 자동 생성을 지원한다.
    - `## 헤더 1{:md}`로 만든 경우, ID는 대체로 `헤더-1`이다.
- 해당 ID를 통해 [내부링크](/posts/markdown/links)를 걸 수 있다.

## 2. Example

- ID 부여 방법
    ```md title="Markdown"
    ## Header {#ID}
    ```
    ```html title="HTML"
    <!-- 마크다운 방법으로 ID 지정이 불가할 때 -->

    <h2 id="ID">Header</h2>
    ```
- 내부 링크 걸기
    ```md title="Markdown"
    [2. Example](#2-example)
    ```
    ```html title="HTML"
    <a href="#2-example">Example</a>
    ```
