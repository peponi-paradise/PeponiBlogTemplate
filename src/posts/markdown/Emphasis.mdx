## 6. Underline

<table>
    <thead>
        <tr>
            <th>Markdown</th>
            <th>HTML</th>
            <th>Render</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>
            ```md title="Markdown"
            <!-- 대체로 불가 -->
            __Under line__
            ```
            </td>
            <td>
            ```html title="HTML"
            <u>Underline</u>

            <!-- 또는 -->

            <span style="text-decoration: underline">
            Underline
            </span>
            ```
            </td>
            <td>
            <span style={{textDecoration: 'underline'}}>
            Underline
            </span>
            </td>
        </tr>
    </tbody>
</table>

## 7. 주의사항

> [!WARNING]
> `밑줄`의 경우 마크다운에서 지원하지 않는 경우가 많다. HTML 스타일로 사용하는 것이 좋다.
>     - `<u>{:html}` 태그의 경우 HTML5에서 철자 오류 또는 주석으로 사용된다.
