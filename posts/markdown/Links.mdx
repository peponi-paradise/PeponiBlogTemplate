## 1. Introduction

- 글에 링크를 다는 기능
- `[링크 이름](링크 주소)` 형식으로 작성

## 3. Link Target

- `Markdown`의 경우 새 창에서 열기 같은 링크 기능을 제공하지 않는다.
- 필요한 경우 직접 `<a>{:html}` 태그를 삽입한다.

<table>
<thead>
        <tr>
            <th>Code</th>
            <th>Render</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>
            ```html title="HTML"
            <a href="https://google.com" target="_blank">Google</a>
            ```
            </td>
            <td>
            <a href="https://google.com" target="_blank">Google</a>
            </td>
        </tr>
    </tbody>
</table>
